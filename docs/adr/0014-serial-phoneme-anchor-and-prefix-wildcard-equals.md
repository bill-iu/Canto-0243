# 串列韻／聲錨與前綴通配等號

領域詞彙：見 [CONTEXT.md](../../CONTEXT.md) § 串列韻／聲錨查詢、前綴通配等號查詢、單字韻錨查詢。延續 [ADR-0004](0004-reference-reading-query-normalize-position-match.md)、[ADR-0012](0012-wildcard-code-anchor-query-syntax-v2.md)。

Grill 共識：E2 硬切換串列掃描；G2 單參考字碼夾仍走碼夾等號；I3 雙通道（串列 + `?{≥2字}=`）；`?{字}=` normalize → `{字}=`。

## 決定

1. **串列掃描（E2）**  
   - 左至右：每位數字一音節碼；`{碼}{字}=` 韻母錨；`{碼}={字}` 聲母錨。  
   - 例：`4困=` 一字；`04困=` 二字；`04困=49倒=` 四字（第 2／4 格韻）。  
   - 聲：`04=困49=倒`。  
   - 串列可含 **`?` 通配單槽**（H1）：`?4困=4潦=9倒=`。  
   - **唔收** `*`／`_`／`%`／`@`（仍走原有分派）。

2. **分派（G2）**  
   - `is_framed_equals_query`（含 `2=我3`）**先於**串列。  
   - **≥2 參考字**串列；單參考字碼夾唔改。

3. **前綴通配等號（I3）**  
   - `?{參考字≥2}=` → 首音節通配 + 等號整段韻模板（`?困潦倒=`）。  
   - `?{≥2字}` 漏尾 `=` → hint。  
   - `?2我=3` 唔收。

4. **Normalize**  
   - `?{單字}=` → `{單字}=`（冗餘 `?`）；規範形 `就=`。

5. **Hint（固定）**  
   - 純漢字多錨：附例 `04困=49倒=`。  
   - `?困潦倒` 漏 `=`：須加 `=`。
