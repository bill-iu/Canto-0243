# 歧義粵拼錨雙列結果與獨立鼻音邊界

領域詞彙：見 [CONTEXT.md](../../CONTEXT.md) § 粵拼錨、獨立鼻音韻母、歧義粵拼錨雙列結果、雙聲母粵拼錨。延續 [ADR-0005](0005-jyutping-anchor-query-syntax.md)。

粵拼字母 `m` 同 `ng` 可以係**輔音聲母**（如「門」`mun`、「我」`ngo`），亦可以係**獨立鼻音韻母**（如唔 `m4`、五 `ng5`）；codec 將後者存入 `initials` 欄，但填詞上**無聲母**。創作者查 `{碼}m{碼}`、`{左}ng{右}`、`?m?`、`?ng?` 時，唔應被迫二選一。我們決定：

1. **歧義觸發清單** — 錨定串含 **`m` 或 `ng` 作碼夾或三格中格**（如 `3m4`、`2ng0`、`?m?`、`?ng?`）→ **歧義粵拼錨雙列結果**；其餘（`3h4`、`3gw4`、`23o`、`?yut?` 等）維持單列。
2. **雙列語意** — 一次查詢並行兩維：**聲母列**（該格有真正輔音 `m` 或聲母叢集 `ng`，如「門」「我」）同 **韻母列**（該格**獨立鼻音韻母** `m`／`ng` 等價，如唔／吾／五／午）。兩列獨立；同一詞條通常只出現喺一列。
3. **獨立鼻音邊界** — 凡粵拼**整個音節**僅為 `m` 或 `ng`（加調）嘅字面，**只**屬韻母維度；比對跟**領域聲韻**，**唔**跟 codec `initials` JSON 字面（避免把唔／五誤當聲母列結果）。
4. **韻母 m≡ng** — 韻母列上獨立 `m` 與獨立 `ng` **完全等價**（mrpinyin M／NG 欄）；聲母列上 `m` 與 `ng` **完全不等價**。
5. **碼夾單字母** — `{首碼}{單字母}{末碼}` 喺**聲母維**含 `m`（如 `3m4` 聲母列→「門」類）；**唔**表示韻母狀態嘅 `m`（該維喺韻母列）。
6. **雙聲母粵拼錨** — `{首碼}{ng|gw|kw}{末碼}` 擴展聲母錨（如 `2ng0` 聲母列→「我」類、`3gw4`→「關」類）；`gw`／`kw` 僅聲母、無歧義雙列。
7. **分頁契約** — 同**近反義關係查詢**：`limit`／`offset` 對**聲母＋韻母合併序列**生效；每項帶 **錨維度** 標籤（如 `anchor_dimension: initial | final`）供前端分節（reuse syn section 模式）。
8. **範圍** — 僅 **0243搜尋模式** 缺字家族；**近反義關係查詢**唔收。

**Considered Options**

- **碼夾二選一** — `{碼}m{碼}` 預設只聲母或只韻母 — 創作者要記兩套寫法或接受查唔到另一維。
- **顯式標記** — `3=m4` 聲母 vs `3m4` 韻母 — 同現有漢字等號語法重疊，教學成本高。
- **跟 codec `initials` 比對** — 實作最省 — 唔／五會錯入聲母列，違反 mrpinyin 同填詞直覺。
- **每列各自 `limit`** — 兩列各 20 條 — UX 清晰但 API／「載入更多」同近反義不一致。
- **合併 `limit`（採用）** — 同 syn 模式；stats 顯示 `聲母 N　韻母 M（已載入 T）`。

**Consequences**

- `query_parse` 須識別歧義粵拼錨查詢類型；`query_dispatch` 雙執行聲母／韻母 match spec，合併結果並標 `anchor_dimension`。
- `jyutping_anchor`／`classify_latin_anchor`：`m` 從「永遠韻母」改為歧義；`parse_code_initial_query` 收 `m`；新增 `{碼}{ng|gw|kw}{碼}` 聲母 parser（優先於碼夾韻母解析）。
- 領域 helper（如音節是否**獨立鼻音**）供聲母列過濾同韻母列匹配；`word_cache` 韻母索引須覆蓋獨立鼻音（`m`／`ng` 雙鍵）。
- 前端 `renderSearchResults`：非 syn 模式下若結果含 `anchor_dimension`，分「聲母」「韻母」兩節（同 `createSynSection`）。
- 測試：`tests/test_standalone_nasal_rhyme.py` 擴充雙列與獨立鼻音邊界；回歸 `3n4` 聲母唔匹配 `ng` 聲母。
- [ADR-0005](0005-jyutping-anchor-query-syntax.md) 第 2 點「`m` 歸韻母粵拼錨」在**歧義查詢**下改為**雙列**；非歧義模板（如 `23o` 末格）不變。
