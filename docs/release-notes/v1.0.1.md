# Canto-0243 v1.0.1

離線粵語填詞查韻工具：依 0243／02493 數字碼、粵拼與韻母規則，幫創作者找可替換詞條與近義／反義關係。  
**Offline Cantonese rhyme workbench** for lyricists: search by 0243 tone codes, Jyutping, and rhyme patterns; find swap candidates and near/antonym relations.

**免安裝**：解壓後雙擊即可，無需自行安裝 Python。  
**Zero-install**: extract and double-click — no Python setup required.

1. 下載 `canto-0243-portable.zip` 並**解壓整個資料夾**  
   Download `canto-0243-portable.zip` and **extract the whole folder**.

## macOS Intel 使用步驟 / macOS Intel quick start

1. 下載 `canto-0243-portable-macos-x86_64.tar.gz` 並解壓  
   Download the tar and extract.

2. 進入 **`canto-0243-portable`** 資料夾  
   Open the **`canto-0243-portable`** folder.

3. 雙擊 **`Canto-0243.command`**（會開 Terminal，屬正常現象）  
   Double-click **`Canto-0243.command`** (Terminal opens — expected).

   Your browser opens the search UI.

**Apple Silicon (M-series)**: no native arm64 build yet; download the **x86_64** tar above — macOS will typically offer **Rosetta 2** translation.

## 本版說明 / Release notes

- **v1.0.1** 為 **v1.0.0** 之詞庫品質修訂：近義橋反義關係重跑並套用品質閘門，**`!` 反義查詢**覆蓋與可靠性明顯改善（例如哀愁、悲傷、憂傷等近義簇可借橋補反義）。  
  **v1.0.1** is a **lexicon quality patch** over **v1.0.0**: near-synonym bridge antonyms were re-ingested with quality gates — **`!` antonym search** is broader and more reliable for near-synonym clusters.

- 免安裝套件與下載方式同 **v1.0.0**（Windows zip + macOS Intel tar，同一 Release 頁）。  
  Zero-install packages and download layout unchanged from **v1.0.0** (Windows zip + macOS Intel tar on this page).

- **Apple Silicon（arm64）** 本版仍未提供原生套件；M 系列 Mac 可先經 **Rosetta 2** 使用 x86_64 tar。  
  **Apple Silicon (arm64)** still has no native build; M-series Macs can use the **x86_64** tar via **Rosetta 2**.

- macOS 套件**未 notarize**；首次啟動需依上方 Gatekeeper 步驟放行一次。  
  macOS builds are **not notarized**; first launch requires the Gatekeeper steps above once.
