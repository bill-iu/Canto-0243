## Canto-0243 v1.0.4-beta

**內部測試用** — SSOT `build-db` 詞條管線 + declarative 詞庫勘誤 overlay（`dev` 分支快照）。

### 資產

- `canto-0243-portable.zip` — Windows 免安裝（`START.bat`）
- `lyrics.db` — 詞條庫（約 25k 詞）
- `words-lexicon.json` — 詞級標音副件

### 測試重點

1. 解壓 zip → 雙擊 `START.bat` → 瀏覽器開查韻頁
2. 基本查詢：單字、詞語、粵拼、0243 碼
3. 近義／反義（若有）：例 `哀愁!`、`悲傷!`
4. 詞庫勘誤是否生效：「不斷」應只有正確編碼列

### 已知限制

- macOS tar **未**包含（僅 Windows beta）
- Apple Silicon arm64 不提供
- 基於 `dev` 分支，**非**正式 `main` 發佈
