# Canto-0243 v1.0.3

離線粵語填詞查韻工具：依 0243／02493 數字碼、粵拼與韻母規則，幫創作者找可替換詞條與近義／反義關係。  
**Offline Cantonese rhyme workbench** for lyricists: search by 0243 tone codes, Jyutping, and rhyme patterns; find swap candidates and near/antonym relations.

**免安裝**：解壓後雙擊即可，無需自行安裝 Python。  
**Zero-install**: extract and double-click — no Python setup required.

## 本版說明 / Release notes

- **查詢語意解釋**：搜尋框下方即時顯示當前查詢意圖（由伺服器解析，與實際分派一致）；等號查詢附押韻標註（單押／雙押…）。  
  **Query semantic explanation**: live intent summary below the search field (server-authoritative); rhyme labels on equals queries.

- **多分頁搜尋回溯**：每個查詢分頁獨立歷史棧；結果頁可點擊返回，唔再與全域瀏覽器歷史混淆。  
  **Per-tab search history**: each query tab keeps its own back stack; result navigation no longer pollutes global browser history.

- **解釋文案**：碼字查詢（如 `23我`）、碼夾等號（如 `32我=0`）會同時顯示碼約束與韻／聲錨；純數字查詢改為「查同{digits}同音嘅字」。  
  **Explain copy**: hybrid code and code-sandwich equals show both digit and rhyme/initial constraints; pure-digit queries use plainer 同音 wording.

- **詞庫**：沿用 **v1.0.2** 詞條庫快照（程式更新、詞庫不變）。  
  **Lexicon**: same word database as **v1.0.2** (app update only).

- **macOS Intel tar**：Windows zip 先上傳；x86_64 tar 由發佈補件跟進同一 tag。  
  **macOS Intel tar**: Windows zip ships first; x86_64 tar follows on the same tag.

- **Apple Silicon（arm64）** 本版仍未提供原生套件。  
  **Apple Silicon (arm64)** still has no native build.

- macOS 套件**未 notarize**；首次啟動需依上方 Gatekeeper 步驟放行一次。  
  macOS builds are **not notarized**; first launch requires the Gatekeeper steps above once.
