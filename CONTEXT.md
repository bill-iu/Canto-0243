# 0243 押韻字典

離線粵語填詞查韻工具：依 0243／02493 數字碼、粵拼與韻母規則，幫創作者找出可替換的詞條與近義／反義關係。

## Language

### 詞條與音韻資料

**詞條**：
字典中一條可查記錄，由字面（一至多個漢字）、0243 碼與粵拼組成；同一字面可有多條詞條（多讀音）。
_Avoid_：Word（實作名）、候選（僅在描述搜尋結果清單時使用）

**字面**：
詞條上顯示的漢字序列（一至多字），例如「香港」的字面長度為 2。
_Avoid_：詞面、字串、char（實作名）

**0243 碼**：
附在詞條上的數字序列；**每一格對應一個音節的聲調**，經固定映射表轉成 0/2/3/4/9 數字（實作：`get_0243_code`，如 tone 2→9、6→2）。長度與字面音節數一致（「香港」兩音節 → 兩位碼）。
_Avoid_：code（實作名）、音碼、把 0243 碼當韻母鍵盤映射

**粵拼**：
詞條的粵語羅馬字標注，一個音節一段（如「香港」→ `hoeng1 gong2`），與 0243 碼逐音節對應。
_Avoid_：jyutping、拼音（易與普通話混淆）

**讀音**：
詞條上**完整音節**的粵拼讀法（如「你」→ `nei5`），含聲母、韻母、聲調。查詢比對時**不必然**拿整段讀音去對：等號碼夾裡的參考字多半只抽出**一個維度**（聲母或韻母）當錨；逐格 **0243 碼數字**則單獨約束該音節的**聲調映射**，與參考字是 `nei5` 還是 `nei1` 無關。
_Avoid_：把參考字的整段讀音等同於該格的查詢條件

**聲母**：
粵拼音節的輔音部分（如 `gong2` 的 `g`）；同聲比對以聲母為單位。
_Avoid_：initial、字頭

**韻母**：
粵拼音節的韻部（含調，如 `gong2` 的 `ong2`）；**同韻比對**以韻母為單位（與 0243 碼的聲調編碼是不同維度）。
_Avoid_：final、字尾

### 比對關係

**同韻**：
兩個字面在對應音節位置上韻母一致；搜尋語法常以 `=` 或參考字標示（如 `?就=`、`香港=`）。
_Avoid_：押韻（指比對規則時）、rhyme

**同聲**：
兩個字面在對應音節位置上聲母一致；搜尋語法常以 `=` 與參考字標示（如 `?=就`、`=香?`）。
_Avoid_：用「同音」泛指同韻或同聲、initial（實作名）

**同音**：
在本工具中，指字面對應音節上的 0243 碼與查詢數字一致（鬆模式下含 0243 等價變體）；不指整個音節聽感完全相同。
_Avoid_：用「同音」泛指同韻或同聲；同碼（若已用同音作 UI 用語則不必並列）

### 搜尋模式

**鬆模式**：
搜尋功能層的預設模式：0243 碼比對時允許 m1 等價變體（API：`mode=m1`；介面顯示「鬆」）。
_Avoid_：m1（實作名，僅程式與 URL）

**緊模式**：
搜尋功能層的進階模式：在 0243 韻母碼基礎上納入 9 鍵聲調區分（API：`mode=m2`；介面顯示「緊」）。
_Avoid_：m2、02493 模式（作為獨立領域詞）

**近反義模式**：
以詞義關係為主的搜尋路徑，回傳近義、反義與語意相關詞條（API：`mode=syn`；介面按鈕顯示「查」）。
_Avoid_：syn、查模式（作為模式名稱）、關係模式

**語意相關**：
與查詢詞在語意上有關聯、但非近義亦非反義的詞條（近反義模式第三類結果）。
_Avoid_：相關詞（不區分類型時）、semantic_related

### 查詢語法

**查詢分派**：
將創作者輸入字串分類為等號查詢、碼字查詢、缺字查詢等語法類型並路由至對應執行路徑的模組邊界（實作：`QueryEngine` / `parse_query`）。分派優先序本身是語意的一部分。
_Avoid_：parser、handler chain（實作細節）

**近反義關係查詢**：
在鬆或緊模式下，以 `~`（近義）或 `!`（反義）查詞義關係的查詢；可帶前綴 0243 碼（如 `33!開心`）。`!!` 表示反義複合詞查詢。
_Avoid_：稱此為「開啟近反義模式」（那是 `mode=syn` 的另一條路徑）

**參考字**：
查詢語法中標定某音節位置的漢字，用於同韻、同聲或字面約束（如 `23就` 的「就」、`2=我3` 的「我」）。
_Avoid_：錨字、錨點字

**字面參考**：
以 `@` 連接參考字，要求該位置字面與參考字完全相同、不做同韻擴展（如 `23@就`）。
_Avoid_：字面錨、@錨

**缺字查詢**：
以 0243 碼與通配符（`?`、`_`、`%` 等價）或漢字，逐位置描述目標字面形狀的查詢（如 `香??`、`23?就`）。
_Avoid_：位置查詢、mask 查詢、wildcard

**碼字查詢**：
以連續 0243 碼開頭、夾或接參考字的查詢（如 `23就`、`23*就`）；前碼段約束同音，參考字約束同韻、同聲或字面。
_Avoid_：混合查詢、hybrid

**等號查詢**：
查詢字面以 `=` 結尾，要求結果與 `=` 前整段字面逐音節同韻（如 `香港=`）；含碼夾參考字變體（如 `2我=3`、`23=你4`）。

碼夾參考字規則（`{左碼}{=?}{參考字}{=?}{右碼}`）：

**兩條獨立約束（必須同時滿足）**：
1. **碼位數字**（來自查詢字串中的 0–9）：逐格約束該音節**聲調**經 `TONE_MAP` 後的 0243 digit。整詞 `Word.code` 須等於 `左碼+右碼`（如 `23=你4` → `234`）。例：`走你好` 讀 `zau2 nei5 hou2` → code `949`，**不是** `234`。
2. **參考字錨**（只取一維，不是整段讀音；**單值**，不對 DB 多詞條做 OR）：
   - `=` 在參考字**左**（如 `23=你4`）→ 該格比**聲母**（「你」只提供 `n`，不是 `nei5` 整節）。
   - `=` 在參考字**右**（如 `23你=4`）→ 該格比**韻母**（「你」只提供 `ei`）。
   - 實作：參考字取詞庫中該字**一條**讀音的聲母／韻母即可（單音節參考字如「你」實務上恆為單值）。

**錨位**：`start_pos = len(左碼) - len(參考字)`（`2=我3` → 第 1 格；`23=你4` → 第 2 格）。

**範例 `23=你4`（鬆模式 m1）**：
- 三字詞，code=`234`；第 2 格（pos 1）碼位數字 = `3`。
- 該格聲調：鬆模式下 digit `3` 接受 **tone 1 和／或 tone 2**（m1 等價：`3`↔`9`）；緊模式 m2 則僅 **tone 1**（digit `3`）。
- 該格聲母：與「你」的聲母相同（`n`），**不**要求該字面上是「你」，也不要求與「你」同韻母或同聲調。

左右碼皆存在時：參考字僅作聲母／韻母錨，**不要求**結果該格字面與參考字相同。
_Avoid_：等號韻（作查詢類型名時）、模板查詢、用參考字整段讀音代替碼位聲調約束

### 詞庫與注入

**詞庫（Lexicon）**：
決定哪些字面是「真詞」、以及詞級讀音（`char` + `jyutping` + `code`）的權威來源。Runtime 的 `words` 表是搜尋用副本；`_ensure` 注入必須經詞庫命中，禁止憑空造字或逐字瞎拼讀音。
_Avoid_：把 pycantonese 猜讀當詞庫、測試用造字進 production ensure

**LexiconPort（計畫介面）**：
可 mock 的詞庫埠，職責分三層：**收錄門檻**（能否注入）、**讀音權威**（jyutping 從哪來）、**排序信號**（搜尋結果誰靠前）。短期實作為 `Static0243Lexicon`（現有 ingest）；長期疊加 [rime-cantonese-upstream](https://github.com/CanCLID/rime-cantonese-upstream)。

**收錄與注入政策（已定）**：
- **多字（len≥2）**：僅當詞庫有該字面的**詞級列**（含 `jyutping` + `code`）才可注入；可有多個 `(char, code)` 列。詞庫未命中 → **不注入**、不用 `pycantonese` 逐字拼接（避免「好」`hou2`/`hou3` 等多音字錯讀）。
- **單字（len=1）**：DB 已有該字列 → 不重複注入。否則 P2 起從 rime `char.csv` 取 `pron_rank=預設` 讀音注入；過渡期見實作路線圖。
- **essay 語料頻次**（[essay-cantonese.txt](https://raw.githubusercontent.com/rime/rime-cantonese/refs/heads/main/essay-cantonese.txt)，`詞\t頻次`）：**不**單獨開注入門、**不**用於猜讀音；僅供收錄後的**排序／常用度**（P3）。
- **pron_rank**（rime `char.csv`）：`預設` > `常用` > `罕見`，用於單字讀音選擇與長期排序；多字讀音以 `word.csv` / `phrase_fragment.csv` 等**詞級標音**為準（含變調後實讀）。

**長期資料來源**：
- [rime-cantonese-upstream](https://github.com/CanCLID/rime-cantonese-upstream)：`char.csv`、`word.csv`、`phrase_fragment.csv` 等（CC BY 4.0，使用時須標註）。
- essay 語料：驗證自然語言常見度（動賓、口語碎片等辭典未必收錄的組合）。

**實作路線圖**：
- **P0**：等號錨位修復、本文件語意、測試（`get_0243_code` 生成 code，禁止手填假碼）。
- **P1**：`LexiconPort` + `Static0243Lexicon`；多字僅詞庫命中可注入。
- **P2**：ingest rime upstream；單字 ensure 改 `char.csv` 預設。
- **P3**：essay 頻次索引 → 搜尋排序。
- **P4**：常用詞 curated 層 + 完整 `pron_rank` 排序策略。

### 使用者

**創作者**：
使用本工具搜韻、換字、查近反義以完成歌詞、文案或其他粵語創作的人。
_Avoid_：填詞者（範圍過窄時）、用户、user
