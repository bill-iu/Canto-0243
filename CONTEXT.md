# 0243 押韻字典

離線粵語填詞查韻工具：依 0243／02493 數字碼、粵拼與韻母規則，幫創作者找出可替換的詞條與近義／反義關係。

## 如何使用本文件

本文件是**領域詞彙表**，不是實作規格。討論功能、測試或文案時，優先使用下列正名；`_Avoid_` 列出易混淆或禁用的說法。

| 章節 | 內容 |
|------|------|
| 核心資料 | 詞條、字面、0243 碼、粵拼、聲母／韻母 |
| 比對關係 | 同韻、同聲、同音 |
| 搜尋模式 | 鬆、緊、近反義 |
| 查詢語法 | 各輸入形式與碼夾等號規則 |
| 詞庫與排序 | 收錄、essay 詞頻、結果誰靠前 |
| 產品邊界 | 離線交付、創作者 |

## 核心資料

**詞條**：
字典中一條可查記錄，由字面、0243 碼與粵拼組成；同一字面可有多條詞條（多讀音）。
_Avoid_：Word（實作名）、候選（僅在描述搜尋結果清單時使用）

**字面**：
詞條上顯示的漢字序列（一至多字），例如「香港」的字面長度為 2。
_Avoid_：詞面、字串、char（實作名）

**0243 碼**：
附在詞條上的數字序列；每一格對應一個音節的聲調，經固定映射轉成 0/2/3/4/9。長度與字面音節數一致（「香港」→ 兩位碼）。
_Avoid_：code（實作名）、音碼、把 0243 碼當韻母鍵盤映射

**粵拼**：
詞條的粵語羅馬字標注，一音節一段（如「香港」→ `hoeng1 gong2`），與 0243 碼逐音節對應。
_Avoid_：jyutping、拼音（易與普通話混淆）

**讀音**：
詞條上完整音節的粵拼（如「你」→ `nei5`）。查詢比對時，參考字多半只提供**聲母或韻母其中一維**作錨；0243 碼數字則單獨約束該格聲調，與參考字是 `nei5` 還是 `nei1` 無關。
_Avoid_：把參考字的整段讀音等同於該格的查詢條件

**聲母**：
粵拼音節的輔音部分（如 `gong2` 的 `g`）；同聲比對以聲母為單位。
_Avoid_：initial、字頭

**韻母**：
粵拼音節的韻部（含調，如 `gong2` 的 `ong2`）；同韻比對以韻母為單位，與 0243 碼的聲調維度不同。
`j-` 後接 `y-` 的音節（如 `jyut`、`jyu`、`jyun`）：聲母為 `j`，`y` 屬韻母一部分（`yut`、`yu`、`yun`），不拆成聲母叢集 `jy`。
_Avoid_：final、字尾

## 比對關係

**同韻**：
兩個字面在對應音節位置上韻母一致；常以 `=` 或參考字標示（如 `?就=`、`香港=`）。
_Avoid_：押韻（指比對規則時）、rhyme

**同聲**：
兩個字面在對應音節位置上聲母一致；常以 `=` 與參考字標示（如 `?=就`、`=香?`）。
_Avoid_：用「同音」泛指同韻或同聲、initial（實作名）

**同音**：
在本工具中，指對應音節上的 0243 碼與查詢數字一致（鬆模式下含等價變體）；不指整個音節聽感完全相同。
_Avoid_：用「同音」泛指同韻或同聲；同碼（若 UI 已用「同音」則不必並列）

## 搜尋模式

**鬆模式**：
預設搜尋模式：0243 碼比對時允許等價變體（介面顯示「鬆」）。
_Avoid_：m1（實作名，僅程式與 URL）

**緊模式**：
進階搜尋模式：在 0243 基礎上納入 9 鍵聲調區分（介面顯示「緊」）。
_Avoid_：m2、02493 模式（作為獨立領域詞）

**近反義模式**：
以詞義關係為主的搜尋，回傳近義、反義與語意相關詞條（介面按鈕顯示「查」）。
_Avoid_：syn、查模式（作為模式名稱）、關係模式

**語意相關**：
與查詢詞在語意上有關聯、但非近義亦非反義的詞條（近反義模式第三類結果）。
_Avoid_：相關詞（不區分類型時）、semantic_related

## 查詢語法

**查詢分派**：
將創作者輸入分類為等號、碼字、缺字等語法類型並路由至對應搜尋路徑；**分派優先序**本身是領域規則的一部分。
_Avoid_：parser、handler chain（實作細節）

**近反義關係查詢**：
在鬆或緊模式下，以 `~`（近義）或 `!`（反義）查詞義關係；可帶前綴 0243 碼（如 `33!開心`）。`!!` 表示反義複合詞查詢。
_Avoid_：稱此為「開啟近反義模式」（那是近反義模式的另一條路徑）

**反義複合詞**：
填詞常用的兩字對舉詞（如「生死」「是非」），由專案 curated 列表定義；`!!` 只回傳該列表中、且詞庫存在的 2 字詞條。
_Avoid_：一般雙字詞、僅因資料管道產生的單字反義 pair

**近義橋反義**（ingest 補全）：
對**有近義、無反義**的字面：在其近義候選中（與 runtime `~` 相同來源：`word_relations` 近義 **＋** static thesaurus），以語意向量找最相似、且本身已有反義的字面作**橋接近義**（候選內 cosine argmax，無額外門檻），將該橋接詞的**全部直接反義**（同 `!橋接詞`、不 expand）借作原字的反義，寫入關係表（`source=ant_syn_bridge`，`score`=W–S cosine）並去重。僅 ingest 階段執行，不於 runtime 載入 embedding。
_Avoid_：反義橋接（作泛稱）、runtime 即時猜反義

**參考字**：
查詢語法中標定某音節位置的漢字，用於同韻、同聲或字面約束（如 `23就` 的「就」、`2=我3` 的「我」）。
_Avoid_：錨字、錨點字

**字面參考**：
以 `@` 連接參考字，要求該位置字面與參考字完全相同、不做同韻擴展（如 `23@就`）。
_Avoid_：字面錨、@錨

**缺字查詢**：
以 0243 碼、通配符（`?`、`_`、`%` 等價）或漢字，逐位置描述目標字面形狀（如 `香??`、`23?就`）。
_Avoid_：位置查詢、mask 查詢、wildcard

**碼字查詢**：
以連續 0243 碼開頭、夾或接參考字的查詢（如 `23就`、`23*就`）；前碼段約束同音，參考字約束同韻、同聲或字面。
_Avoid_：混合查詢、hybrid

**等號查詢**：
查詢字面以 `=` 結尾，要求結果與 `=` 前整段字面逐音節同韻（如 `香港=`）。
_Avoid_：等號韻（作查詢類型名時）、模板查詢

**碼夾等號查詢**：
形如 `{左碼}{=?}{參考字}{=?}{右碼}` 的等號查詢（如 `2=我3`、`23=你4`、`34=我`、`03=催是`）。須**同時**滿足兩類約束：(1) 整詞 0243 碼等於左碼＋右碼（無右碼時整詞 code 即左碼）；(2) 參考字在錨位只提供**一維**——`=` 在左比聲母、`=` 在右比韻母，不要求結果該格字面是參考字本身。錨位：`len(左碼) − len(參考字)`（`23=你4` → 第 2 格比聲母「你」的 `n`；`34=我` → 二字、code `34`、末格同聲「我」；`03=催是` → 二字、code `03`、整詞同聲「催是」，與詞庫詞 `03=隨時` 等價若音節拼接讀音一致）。三字同聲請用碼字查詢 `34*=我`，勿與碼夾等號混淆。
_Avoid_：用參考字整段讀音代替碼位聲調約束、對多讀音參考字做 OR 比對

## 詞庫與排序

**詞庫**：
決定哪些字面是「真詞」、以及詞級讀音（字面＋粵拼＋0243 碼）的權威來源。搜尋用的詞條庫是詞庫的副本；注入新詞必須經詞庫命中，禁止憑空造字或逐字拼接讀音。
_Avoid_：把自動猜讀當詞庫、測試用造字進正式收錄

**詞庫埠**：
詞庫的抽象邊界，分三層職責：**收錄門檻**（能否注入）、**讀音權威**（粵拼從哪來）、**排序信號**（結果誰靠前）。
_Avoid_：把 runtime 詞條庫本身稱作詞庫

**多字收錄**：
長度 ≥ 2 的字面，僅當詞庫有該詞的**詞級標音**（粵拼＋0243 碼）才可注入；可有多個讀音列。詞庫未命中 → 不注入。
_Avoid_：逐字拼接多音字讀音後注入

**音節拼接讀音**：
詞庫無該多字詞級標音時，逐字取 rime 單字「預設」讀音拼成整詞粵拼與 0243 碼；用於查詢、結果頁顯示與參考字錨點解析，**不**替代詞庫收錄門檻，**不**宣稱為詞級標音。任一字缺 rime 預設則拼接失敗；單字有多個 rime 讀音時**只取預設**，不展開讀音組合。
_Avoid_：與詞級標音混稱、把拼接結果當詞庫權威、線上 API 猜音

**單字收錄**：
長度 1 的字面，詞條庫已有則不重複注入；否則從 rime 單字表取「預設」讀音注入。
_Avoid_：為排序而單獨收錄 essay 裡出現但未收錄的字

**Essay 語料**：
粵語書面／口語詞頻語料（[rime-cantonese](https://github.com/rime/rime-cantonese) · [essay-cantonese.txt](https://github.com/rime/rime-cantonese/blob/main/essay-cantonese.txt)，CC BY 4.0），格式為「詞＋詞頻」。**不**開注入門、**不**用於猜讀音；僅供排序。語料隨 repo 提供。
_Avoid_：把語料當詞庫、把語料當近義來源

**Essay 詞頻索引**：
啟動時自 Essay 語料建立的**字面→詞頻**查表，僅供搜尋排序；載入記憶體，不寫入詞條庫、不另建詞頻表。更新語料檔後重啟即生效。
_Avoid_：詞頻表（作資料庫實體）、把詞頻存進詞條列、用 vector embedding 取代詞頻排序

**Curated 常用詞**：
專案維護的填詞常用字面列表，在 essay 詞頻相同時作排序加成。
_Avoid_：把 curated 排在 essay 之前（現行 tier 為 essay 優先）

**pron_rank**：
rime 單字表的讀音常用度標記（預設＞常用＞罕見），用於單字讀音選擇與排序 tie-break；多字讀音以詞級標音為準。
_Avoid_：把 pron_rank 當收錄門檻

**搜尋結果排序**：
扁平詞條清單在同一匹配層級內，共用排序信號：**純漢字字面 → essay 詞頻 → curated → pron_rank → 字面**。Code header、近反義分組、缺字查詢的字面優先 tier 等結構性區塊順序不變，只在區塊內套用上述信號。目標是貼近 0243 常用度體感，不要求與 0243.hk 逐詞一致。
_Avoid_：預設字母序、每種語法各一套 tie-break、混雜拉丁字面與純漢字無區分、維護 0243 完整快照作 CI 門檻

## 產品邊界

**離線單機交付**：
本工具的主要形態：創作者在本機完整使用查韻、搜尋與近反義，詞庫與快取皆在本機，不依賴常駐遠端服務。
_Avoid_：把可選自架服務當與離線版並重的一級產品

**自架共用服務**：
多人透過遠端 API 共用同一詞庫的部署形態；**非**目前主要交付目標；相關程式屬凍結、不主動投資。
_Avoid_：Postgres 版、hosted 版（作產品名稱時）

**創作者**：
使用本工具搜韻、換字、查近反義以完成歌詞、文案或其他粵語創作的人。
_Avoid_：填詞者（範圍過窄時）、用户、user

**查韻首屏**：
創作者開啟工具後，首屏以**搜尋列與模式切換**為中心；不展示行銷型 hero、英文 eyebrow 或長說明段。語法教學經頂欄入口或輸入提示進入，不佔首屏垂直空間。
_Avoid_：landing page 式大標、把教學文案堆在搜尋框上方

**介面色調**：
工具介面以**冷灰單色底**與 zinc 階文字為主，搭配**單一藍色 accent** 標示互動與焦點；不走暖紙／琥珀 craft 色板。
_Avoid_：把查韻工具做成 premium-consumer landing 配色

**模式切換**：
鬆、緊、查三種搜尋模式以頂欄 **segmented control**（並排 pill、當前高亮）呈現，一鍵切換；搜尋教學維持獨立入口，不藏進模式 menu。
_Avoid_：把三種主模式藏進下拉選單

**介面主題**：
工具介面支援 **light / dark**，預設跟隨系統 `prefers-color-scheme`；兩種主題皆維持冷灰底、單一藍 accent 與功能優先布局。
_Avoid_：僅 light、或 section 中途反轉主題
