Search manual

所有搜尋語法

每個例子都可以直接執行。點擊後會回到搜尋頁、套用建議模式，並送出查詢。口訣：= 在錨字後 → 韻母；在錨字前 → 聲母（一般查詢與 * 延伸段相同）。

基本查詢

輸入中文字、詞語、數字或粵拼片段。適合先粗略找候選，再點擊結果回查。

0243 / 02493 數字

鬆 使用 0243；緊 使用 02493。純數字會列出符合該數字的候選。

字面位置

漢字直接指定該位置的字面。用 ?、_ 或 % 表示不限位置。

數字 + 尾字（2 字 vs 3 字）

查尾韻用 0243+漢字；只要字面尾字、不要同韻其他字，用 0243+@+漢字。新增漢字位置用 * 連接，例如 23*就（可直打 URL，不需編碼）。

韻母錨點

錨字緊貼 = 右側（錨字=），該位置韻母與錨字相同。錨字本身不必出現在結果中。

聲母錨點

錨字緊貼 = 左側（=錨字），該位置聲母與錨字相同。不可裸寫 =就，須寫 ?=就 或 23*=就。

右 = 查韻母

錨字在 = 左側（錨字=），該位置韻母與錨字相同。整詞模板在詞尾加 =。

左 = 查聲母

錨字在 = 右側（=錨字），該位置聲母與錨字相同。與右 = 查韻母不同，勿混用。

萬用字元

用數字搭配萬用字元?、_ 或 %描述部分已知的位置。

近義 / 反義

~ 查近義，! 查反義。

反義複合詞

!! 會找兩個單字互為反義的二字複合詞；後面接一個字時，會搜尋尾字韻母。
