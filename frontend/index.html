所有搜尋語法

每個例子都可以直接執行。點擊後會回到搜尋頁、套用建議模式，並送出查詢。口訣：= 在錨字後 → 韻母；在錨字前 → 聲母（一般查詢與 + 延伸段相同）。

基本說明

打數字，搵 0243 碼同音嘅詞條。每個數字對應一個音節嘅聲調（0243 碼），查詢有幾個數字就搵幾個音節嘅詞。例如打「23」，會搵整詞 0243 碼同「23」同音嘅二字詞，包括「自己」、「第一」、「做好」。打「232」會搵三字同音詞，例如「是不是」、「自己做」、「沒關係」。0243模式用 0243 碼；02493模式用 02493，調值更細（分清二聲）。

打漢字，查詞條讀音同編碼。例如打「開心」、「明白」、「食飯」，會見到粵拼、0243 碼，以及同音候選。切換近反義模式，可以直接列出該詞嘅近義、反義同語意相關字。

打數字加錨字，逐格約束韻或聲。串列掃描：每位數字一音節碼，{碼}{字}= 韻錨、{碼}={字} 聲錨。例 23就=（二字尾格同「就」韻）、04困=49倒=（四字韻錨）。加槽用 +，如 23+就=（三字尾格同韻）；尾字字面固定用 23@就。

缺字同加長位置。通配用 ?／_／%；加槽用 +，例 +香??、23+就、?30+人。輸入 * 仍接受（等同 +）。

打幾個漢字加等號，搵「同韻」或「同聲」。打「開心=」會搵整詞逐音節同「開心」同韻嘅詞。打「=最好」會搵整詞逐音節同「最好」同聲嘅詞。單格聲母錨打「=我?」、「?=就」。碼夾等號如「2我=3」、「2=我3」，可同時約束 0243 碼同一格聲或韻。

打一個漢字加等號，搵「同韻」或「同聲」詞。打「就=?」會搵首字同「就」同韻嘅二字詞；打「就=」會搵同「就」同韻嘅單字。聲母錨打「=就?」、「?=你」。口訣：「=」在錨字後比韻母，在錨字前比聲母。

打粵拼，精準搵漢字。冇聲調如「syut」，搵粵拼相同嘅單字（忽略聲調）；有聲調「ming4 baak6」會搵準確讀音嘅字「明白」。近反義模式唔收粵拼，請改打漢字或切換 0243模式／02493模式。

打粵拼錨，唔使打漢字參考字。規範形如 ?+yut?、?+hon、3+ngo4、23o（二字韻母）／23+o（三字碼尾韻母）。?+m?、3m4 會分聲母／韻母兩列。

近義、反義同複合詞（0243模式／02493模式）。打「~開心」搵同「開心」近義嘅詞；打「!開心」搵反義詞。可加前綴0243碼，例如「33!開心」。打「~~」搵二字近義複合（如「朋友」「恐懼」）；打「!!」搵二字反義複合（如「生死」「是非」）。語法對稱：~~／33~~／~~你／33~~你，與 !! 系列相同。以上複合詞查詢不適用近反義模式。

基本查詢

漢字、詞語、0243 碼或粵拼。

0243 / 02493 數字

純數字搵同碼詞條；02493 分清二聲。

串列韻／聲錨

連續數字：每位一音節碼。{碼}{字}= 比韻；{碼}={字} 比聲。= 永遠喺參考字右側。

四字部分韻／聲錨

? 標通配格；其餘漢字格逐格比韻／聲（結果唔使同骨架逐字相等）。

前綴通配等號

第 1 格完全通配；其餘音節逐格同參考模板（須尾 = 表韻）。

通配碼錨

首音節 ? 通配，後接連續碼；尾漢字係韻參考字。加槽用 +。

缺字／音查詢（遮罩）

漢字固定字面，數字固定碼，其餘用 ?／_／%。頭格的 + 可省略。

加號錨（+）

+ 連接碼同錨字，標明邊一格。錨字= 同韻母；+=錨字 同聲母；無 = 則字面固定。輸入 * 等同 +。

同韻／同聲錨（=）

錨字= 比韻母，=錨字 比聲母；錨字唔一定出現喺結果。

粵拼錨

缺字族用拉丁拼標韻母、音節或聲母；slot 連接用 +（如 ?+hon、3+ngo4）。

整詞／碼夾等號（=）

詞尾 = 比整詞韻，詞首 = 比整詞聲；碼夾約束單格。{左碼}{全詞}=（碼長=字數）再加 0243 碼約束。

近義 / 反義

~ 近義、! 反義；可加碼前綴。僅 0243／02493 模式。

近義複合詞

~~ 搵二字近義複合；可加碼前綴或尾韻字。

反義複合詞

!! 搵二字反義複合；可加碼前綴或尾韻字。

同音節疊字

$$ 搵二字詞且兩字音節相同（聲調不限）；語法鏡像 ~~。

同音異讀

{左碼}/{右碼} 搵同一字面、至少兩個唔同讀音；? 通配碼位。

連接詞複合詞

中格填詞連接詞（與、和、或…）嘅三字複合；~與~ 近義、!與! 反義。

關係補錄

為已收錄字面補近義或反義關係。種子字面為擴展起點；對端字面提供一跳鄰居來源。

擴展從此字面出發（例如要補「快樂」的關係時填快樂）

關於 Canto-0243

ONE·搵·韻 — 離線粵語填詞查找工作台。

簡介

填粵語歌詞時，往往要在同音、押韻、近義之間快速換字，又要對準 0243 與粵拼讀音。Canto-0243 用 0243／02493 數字碼、粵拼、韻母／聲母規則與近義／反義關係，在幾秒內列出符合條件的詞條；套件解壓即用，詞庫與近反義資料存於本地，無需連網。
