所有搜尋語法

每個例子都可以直接執行。點擊後會回到搜尋頁、套用建議模式，並送出查詢。口訣：= 在錨字後 → 韻母；在錨字前 → 聲母（一般查詢與 * 延伸段相同）。

基本說明

打數字，搵粵語啱音字。例如打「23」，會搵到聲調序列一樣嘅二字詞，包括「做好」、「做就」。打「232」會搵三字同調詞，例如「做就就」、「做得就」。0243模式用 0243 碼；02493模式用 02493，調值更細（分清二聲）。

打漢字，查詞條讀音同編碼。例如打「開心」、「明白」、「食飯」，會見到粵拼、0243 碼，以及同音或相關候選。切換近反義模式，可以直接列出該詞嘅近義、反義同語意相關字。

打數字加尾字，搵啱音押韻嘅詞。譬如打「23就」，會搵頭兩音節同「23」、尾字同韻「就」嘅二字詞。只要尾字字面、唔要同韻擴展，打「23@就」。

可以夾多個韻字或加長位置。例如「23*就」（三字，頭兩音節同「23」，尾字係「就」）、「23?就」（四字碼字混合）。用「?」、「_」或「%」表示該格字面不限，例如「做??」、「?你?」。

打幾個漢字加等號，搵「多字韻」或「同聲」。打「開心=」會搵整詞逐音節同韻「開心」嘅詞。打「=做好」會搵整詞逐音節同聲「做好」嘅詞。單格聲母錨打「=我?」、「?=就」。碼夾等號如「2我=3」、「2=我3」，可同時約束 0243 碼同一格聲或韻。

打一個漢字加等號，搵「同韻」或「同聲」詞。打「就=?」會搵首字同「就」韻嘅二字詞；打「?就=」會搵尾字同「就」韻嘅詞。聲母錨打「=就?」、「?=你」。口訣：「=」在錨字後比韻母，在錨字前比聲母。

打粵拼，精準搵漢字。無調打「syut」搵單字「雪」；打「nei hou」搵「你好」。有調打「ming4 baak6」搵「明白」嘅準確讀音。近反義模式唔收粵拼，請改打漢字或切換 0243模式／02493模式。

近義、反義同複合詞（0243模式／02493模式）。打「~開心」搵與「開心」近義的詞；打「!你」搵反義（含鏡像近義）。可加前綴碼，例如「33!開心」。打「~~」搵二字近義複合（如「朋友」「恐懼」）；打「!!」搵二字反義複合（如「生死」「是非」）。語法對稱：~~／33~~／~~你／33~~你，與 !! 系列相同。以上複合詞查詢不適用近反義模式。

基本查詢

輸入中文字、詞語、數字或粵拼查詢。適合先找候選，再點擊結果回查。

0243 / 02493 數字

0243模式 使用 0243；02493模式 使用 02493。純數字會列出符合該數字的候選。

字面位置

漢字直接指定該位置的字面。用 ?、_ 或 % 表示不限位置。

數字 + 尾字（2 字 vs 3 字）

查尾韻用 0243+漢字；只要字面尾字、不要同韻其他字，用 0243+@+漢字。新增漢字位置用 * 連接，例如 23*就（可直打 URL，不需編碼）。

韻母錨點

錨字緊貼 = 右側（錨字=），該位置韻母與錨字相同。錨字本身不必出現在結果中。

聲母錨點

錨字緊貼 = 左側（=錨字），該位置聲母與錨字相同。不可裸寫 =就，須寫 ?=就 或 23*=就。

右 = 查韻母

錨字在 = 左側（錨字=），該位置韻母與錨字相同。整詞模板在詞尾加 =。

左 = 查聲母

錨字在 = 右側（=錨字），該位置聲母與錨字相同。與右 = 查韻母不同，勿混用。

萬用字元

用數字搭配萬用字元?、_ 或 %描述部分已知的位置。

近義 / 反義

~ 查與該詞近義的候選；! 查反義（含鏡像近義）。可加 0243 碼前綴。僅 0243模式／02493模式；切換近反義模式時改打漢字查整欄關係。

近義複合詞

~~ 找填詞常用的二字近義複合：兩字在字級上互為近義（如「朋友」「恐懼」），只回傳詞庫已有的二字詞。可加碼前綴或尾韻字，語法與 !! 對稱。僅 0243模式／02493模式。

反義複合詞

!! 找兩字互為反義的二字複合詞（如「生死」「是非」）；後面接一個字時，會搜尋尾字韻母。僅 0243模式／02493模式。
